브루잉 커피

* 아래 표의 그라인더가 아닌 경우에는 추출 시간에 맞게 분쇄도를 조정해 주세요. 추출 시간이 너무 빠르면 조금 더 가늘게, 너무 느리면 더 굵게요.

* 분쇄도와 추출 시간은 물을 붓는 속도에 따라 조금 차이가 있을 수 있으니, 너무 연하면 분쇄도를 조금 더 가늘게, 너무 진하면 분쇄도를 더 굵게 조정해 주세요.

HOT

커피 분쇄도 가이드 코만단테 그라인더 바라짜 엔코 그라인더 추출시간
블랙수트 22~24 19~21 2:15~3:00
벨벳화이트 23~25 18~20 2:45~3:00
과테말라 라 몬타냐 22~24 17~19 2:05~2:45
콜롬비아 비야로조 타비 23~25 15~20 2:10~2:50
에티오피아 딤비라 24~26 19~21 2:20~3:00
브라질 알미르 파베루 24~26 19~21 2:30~3:10
    1인분

    원두 20g : 93℃ 물 320g

  • 60g의 물을 나선형을 그리며 부어주세요.
  • 30초 후, 140g의 물을 같은 방법으로 부어주세요.
  • 부었던 물이 모두 떨어지면 120g의 물을 부어주세요.
    2인분

    원두 30g : 93℃ 물 480g

  • 80g의 물을 나선형을 그리며 부어주세요.
  • 30초 후, 200g의 물을 같은 방법으로 부어주세요.
  • 부었던 물이 모두 떨어지면 200g의 물을 부어주세요.

    • *1인분 HOT기준 약 25% 더 굵게 분쇄해 주세요.
    (예. 1인분 HOT 분쇄도가 20이라면, 20 X 1.25 = 25)

ICED

커피 분쇄도 가이드 코만단테 그라인더 바라짜 엔코 그라인더 추출시간
블랙수트 16~18 14~16 2:00~2:30
벨벳화이트 17~19 13~15 2:10~2:40
과테말라 라 몬타냐 16~18 13~15 2:00~2:20
콜롬비아 비야로조 타비 17~19 14~16 1:45~2:05
에티오피아 딤비라 18~20 14~16 2:00~2:20
브라질 알미르 파베루 18~20 14~16 2:10~2:30
    1인분

    원두 20g : 93℃ 물 200g

  • 추출 전, 얼음 50~60g을 서버에 넣어 주세요.
  • 60g의 물을 나선형을 그리며 부어주세요.
  • 30초 후, 80g의 물을 같은 방법으로 부어주세요.
  • 부었던 물이 모두 떨어지면 60g의 물을 부어주세요.
  • 추출된 커피를 얼음컵에 옮겨 맛있게 드셔주세요!

    • *분쇄도는 HOT 기준 약 25% 더 가늘게 분쇄해 주세요.
    (예. HOT 분쇄도가 20이라면, ICED는 20 X 0.75 = 15)

    2인분

    원두 30g : 93℃ 물 300g

  • 추출 전, 얼음 약 100g을 서버에 넣어 주세요.
  • 80g의 물을 나선형을 그리며 부어주세요.
  • 30초 후, 110g의 물을 같은 방법으로 부어주세요.
  • 부었던 물이 모두 떨어지면 110g의 물을 부어주세요.
  • 추출된 커피를 얼음컵에 옮겨 맛있게 드셔주세요!
    핸드드립 분쇄도로 주문한 고객

    원두 25g : 93℃ 물 250g

  • 추출 전, 얼음 50~60g을 서버에 넣어 주세요.
  • 70g의 물을 나선형을 그리며 부어주세요.
  • 30초 후, 100g의 물을 같은 방법으로 부어주세요.
  • 부었던 물이 모두 떨어지면 80g의 물을 부어주세요.
  • 추출된 커피를 얼음컵에 옮겨 맛있게 드셔주세요!

    • *분쇄도는 HOT RECIPE와 동일합니다.

에스프레소 커피

  • 분쇄한 원두를 큰 컵 또는 스팀피처에 넣어 잘 흔들어 주세요.
  • 커피 가루를 포터필터 바스켓에 담고 손가락 등을 이용해 평평하게 만들어 주세요.
  • 탬퍼를 이용해 커피를 수직으로 눌러준 후, 포터필터를 머신에 체결해 주세요.
  • 저울을 이용해 각 커피 레시피에 해당하는 무게의 에스프레소를 추출해 주세요.
  • 목표로 한 추출시간보다 빠르게 추출이 되면 분쇄도를 조금 더 가늘게 조정해 주세요.(더 느리게 추출이 되면 분쇄도를 조금 더 굵게 조정해 주세요.)
