# 빈브라더스 리뉴얼

리뉴얼 대상 사이트 : [빈브라더스](https://beanbrothers.co.kr/, "beanbrothers link")

### 프로젝트소개

개발기간 : 2023.11.13~2023.12.1

개발환경:
Visual Studio Code을 이용하여 개발되었고 시안은 figama을 사용했습니다

고민사항 및 해결과정

1. 고객이 정보를 비교하기 쉽게 배치
2. 반응형 구조로 재배치
3. 메뉴를 관련성있게 재구성
4. 커피추출법 페이지를 정리된 형식으로 작성
